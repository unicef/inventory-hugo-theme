Note: This page is used for two purposes:

  1. For Hugo webmasters: See examples of how the theme renders different kinds of content. It shows most of the elements and various markup that can be used across the theme.

  2. For theme developers: A quick and fast way to see many elements on a single page at once. This is helpful for testing for regressions when introducing new front-end changes.

The text used for the remainder of the article is default "lorem ipsum" text. It has no meaning and is used only to show different features.
