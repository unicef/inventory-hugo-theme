Customization

Dot has been built to be as configurable as possible.

Change the logo ðŸ”—

In config.toml you will find a logo variable. you can change your logo there.

The size of the logo will adapt automatically

Change the favicon ðŸ”—

If your favicon is a png, just drop off your image in your local static/images/ folder and name it favicon.png

If you need to change this default behavior, create a new file in layouts/partials/ named head.html. Then write something like this:

1<link rel="shortcut icon" href="/images/favicon.png" type="image/x-icon" />

Change default colors ðŸ”—

Dot support change color. You can change the colors from config.toml. You can change the colors of the template as you want.

1[params]
2  # Change default color scheme with a color name or color code.
3  primary_color = "#02007e"
4  body_color = "#f9f9f9"
5  text_color = "#636363"
6  text_color_dark = "#242738"
7  white_color = "#ffffff"
8  light_color = "#f8f9fa"

Change default Fonts ðŸ”— 

1[params]
2  # font family
3  font_family = "lato" # Choose font family from : https://fonts.google.com/

Add Author byline ðŸ”—

We can add author byline to an article by adding author and authorEmail attributes to the frontmatter of that article page. Here is an example -

1author: John Doe
2authorEmail: john@example.com
